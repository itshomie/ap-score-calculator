# AP Score Calculator

Independent AP score calculator for practice-exam study planning.

Live site: [AP Score Calculator](https://apscorecalculator.app/)

APScoreCalculator.app helps students, parents, tutors, and teachers estimate a practice-test result from multiple-choice and free-response raw scores. It is browser-based, free to use, and does not require an account.
